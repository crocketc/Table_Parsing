## [0.1.0] - 2026-05-03

### 新增

#### 核心功能
- **统一解析接口**: 提供 `parse_file()` 函数，支持 CSV、XLS、XLSX 格式
- **中间表示（IR）**: 完整的数据模型（Cell、Sheet、Workbook）
- **自动格式检测**: 根据文件扩展名自动选择合适的解析器
- **智能编码检测**: 使用 charset-normalizer 自动检测文件编码
- **自动分隔符检测**: 支持逗号、分号、Tab、管道符等分隔符

#### CSV 解析器
- 自动编码检测（GBK、UTF-8、UTF-16 等）
- 自动分隔符检测
- 智能数据类型推断（整数、浮点数、布尔值、日期）
- 分块解析支持，可处理大文件
- 容错机制，单个单元格错误不影响整体解析

#### Excel 解析器
- **XLSX 解析器**:
  - 完整的工作表、单元格数据读取
  - 公式提取和保留
  - 合并单元格检测
  - 隐藏行/列识别
  - 数字格式和样式信息
  - 内嵌媒体（图片、图表）提取

- **XLS 解析器**:
  - 兼容旧版 Excel 文件
  - 数据类型映射（XLNumber → Python 数值）
  - 合并单元格支持
  - 公式单元格识别
  - 工作表元数据提取

#### 数据模型
- **MediaObject**: 支持图片和图表媒体对象
- **Cell**: 完整的单元格信息（值、类型、公式、样式、合并信息）
- **Sheet**: 工作表数据（名称、可见性、维度、单元格矩阵）
- **Workbook**: 工作簿容器（元数据、工作表列表）

#### 高级功能
- **媒体提取**: 从 XLSX 文件中提取图片和图表
- **AI 模型集成**: ModelClient 协议支持 OpenAI 兼容 API
- **媒体理解**: AI 驱动的图片和图表内容理解

#### 开发工具
- **质量检查脚本**: `scripts/check_quality.py`
- **验收测试**: 端到端功能和性能测试
- **测试覆盖率**: 82%+ 代码覆盖率
